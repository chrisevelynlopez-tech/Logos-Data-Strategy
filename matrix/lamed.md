# FICHA TÉCNICA ATÓMICA: ל (Lamed)
# ID_LOGOS: 012
# DATACIÓN: 1800 a.C. (Inscripciones de Serabit el-Khadim)

## 1. DIMENSIÓN FÍSICA Y ANATÓMICA
* **Pictografía:** Un aguijón o vara de pastor (palo con punta o curva).
* **Lógica de la Era:** Herramienta manual indispensable para dirigir el camino del buey durante el arado.
* **Función Natural:** **Dirección / Guía**. El objeto físico que se usa para dar sentido y orientación a la fuerza aplicada.

## 2. COMPORTAMIENTO DEL DATO
* **Identificador:** Sonido inicial del objeto (L-md).
* **Uso técnico:** Identifica el componente que orienta, dirige o controla la aplicación de la fuerza dentro de una secuencia.

## 3. FIABILIDAD ARQUEOLÓGICA
* **Fuente:** Proto-Sinaítico grabado en las minas de turquesa.
* **Contexto:** Bronce Medio. Basado en el uso de herramientas de control para el manejo de ganado y el movimiento ordenado de transporte.
